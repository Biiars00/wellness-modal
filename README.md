# Wellness Modal

## Sobre o projeto

O **Wellness Modal** é uma página de estruturação de modal, cujo tema de desenvolvimento é sobre saúde e bem-estar.

-   Modal dinâmico com Javascript
-   Design responsivo
-   Modo dark e light
